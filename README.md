## h5在线编辑平台

### 技术架构
> 前端：vue2.0 + vue-router + vuex + axios + Element ui

> 后端：nodejs + express + mongoose

### 运行方法
> npm install  // 安装依赖

> npm run webapp // 启动前端工程

> npm run local //  启动后端工程
